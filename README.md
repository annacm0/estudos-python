# Calculadora de Idade em Python 

Este é um projeto simples desenvolvido para praticar a lógica de programação e a manipulação de entradas e saídas de dados com **Python**.

## Como funciona
O programa solicita o nome e o ano de nascimento do usuário, realiza o cálculo com base no ano atual (2026) e exibe a idade aproximada.

## Tecnologias utilizadas
- **Linguagem:** Python
- **IDE:** VS Code
